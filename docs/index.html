About

  • A simple tool to display Session Descrition Protocol (SDP) text in a more readable format.
  • All processing is done in your browser

Tips

  • Add a space before a line in the SDP text to hide it from the output.
  • Conversely, any lines begining with whitespace will not parse and won't display in the output.

What is SDP?

  • SDP is found in many Real Time Communication (RTC) protocols like WebRTC.
  • SDP on Wikipedia
